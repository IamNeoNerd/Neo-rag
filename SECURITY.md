## Reporting a Vulnerability

We take the security of Neo-rag seriously. If you have discovered a security vulnerability, please do not disclose it publicly.

### How to Report

Please report vulnerabilities by opening a **Security Advisory** in the GitHub repository or contacting the maintainers directly.

We will review the issue and respond as soon as possible.
